Listado de herramientas

Listado de herramientas para la mejora de productividad, contiene todas las herramientas necesarias para tratamientos masivos de datos y gestión de documentos de forma web.

ID Nombre Función Enlace
01 Página de Contraseñas Protege enlaces con contraseñas, estos no se almacenan en servidores y no dejan rastro
02 Editor Meta Datos Editor de datos de un archivo, como fecha, hora, editor y propiedad de documentos.
03 Convertidor PDF Fiel convertidor de PDF a otros formatos, este suele ser ideal para convertir pdf a word.
04 Herramientas PDF A diferencia del anterior, esta herramienta es ideal para la compresión y opciones masivas de documentos PDF.
